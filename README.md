# Bioactive Compounds in Acalypha indica for Cervical Cancer Inhibition

## Overview
Cervical cancer remains one of the leading causes of cancer-related 
mortality among women globally, with HPV-associated protein targets 
presenting viable candidates for phytochemical-based drug discovery. 
This project investigates the anticancer potential of bioactive 
compounds isolated from Acalypha indica — a medicinally significant 
plant in the Euphorbiaceae family — through an integrated 
computational and experimental approach.

## Objectives
- Identify and characterise bioactive phytochemical compounds 
  present in Acalypha indica via TLC and GC-MS analysis
- Perform molecular docking simulations against key cervical cancer 
  target proteins to evaluate binding affinities and interaction modes
- Profile candidate compounds for drug-likeness using ADME prediction 
  tools (Lipinski's Rule of Five, toxicity screening)
- Correlate computational predictions with phytochemical screening 
  results to shortlist lead compounds for further study

## Methodology
| Step | Tool / Technique |
|------|-----------------|
| Phytochemical screening | GC-MS |
| Protein structure retrieval | RCSB Protein Data Bank (PDB) |
| Ligand preparation | PubChem, ChemDraw |
| Molecular docking | AutoDock Vina |
| ADME profiling | SwissADME|
| Visualisation | Chimera |
| Data analysis | Microsoft Excel |
